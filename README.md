# Oracle To Postgres

Este projeto está usando Vue 3 + Vite

# Requisitos:
__npm__: 8.5.5
__pnpm__: 7.5.0
__Node__: v16.17.0
Provavelmente não pega ou da algum erro com alguma versão menor que essa.

# Estrutura:
->View:  pastas com as paginas e subpanigas do sistema
->Components: aqui vc cria todo os Components para serem reutilizados
Por favor, nao criem um formulário com as tags do html direto no form, procurar sempre criar um componente customizavel
para cada tipo de input. Tem uma pasta chamada 'Form' onde vc vai colocar cada elemento de form la dentro
Caso o componente nao seja para um formulário crie uma pasta, mas nao misture com componentes de outras finalidades.

Mudou algumas coisas de sintaxe do vue 2 para o 3 entao é bom ver a documentação
Qualquer dúvida: https://vuejs.org/guide/quick-start.html
